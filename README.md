# kivy-custom-webview
Using android webview in Kivy (KivyMD) with some modification.  

  
Kivy v2.2.0  
Python v3.10.11  
KivyMD 1.2.0.dev0  

### SIMPLE WEBVIEW: _simple_webview  
- current page favicon updated (search_favicon)
- current page url updated (search_input)
